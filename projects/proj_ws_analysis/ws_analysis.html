Franka Research 3 (FR3) Workspace Analysis

Reachable and Manipulability Workspace Analysis of the Franka Research 3 Manipulator using ROS2, Pinocchio and Monte Carlo Sampling.

Introduction

A robot manipulator's workspace defines the region in which its end-effector can operate. Workspace analysis is an essential step in robot design, task planning, and workcell design because it determines the capability of a manipulator to perform a given task.

Overview

  • Robot: Franka Research 3 (FR3) 7-DOF Manipulator
  • Framework: ROS2 Jazzy
  • Kinematics Library: Pinocchio
  • Sampling Method: Monte Carlo Joint-Space Sampling
  • Samples Generated: 50,000 Random Configurations
  • Visualization: PyVista
  • Workspace Metric: Yoshikawa Manipulability Index (Translational)

In this project, the workspace of the Franka Research 3 (FR3) manipulator is analyzed using a URDF-based robot model, Pinocchio for kinematic computation, and Monte Carlo joint-space sampling.

Reachable Workspace

Concept

The reachable workspace represents all possible positions that the robot's end-effector can achieve while satisfying its joint limits.

WR = { x ∈ ℝ³ | ∃ q ∈ Q such that x = f(q) }

where,

  • q represents the joint configuration.
  • Q denotes the set of all valid joint configurations within the joint limits.
  • f(q) is the forward kinematics mapping from joint space to Cartesian space.

Implementation

Computing the exact reachable workspace of a redundant manipulator analytically is computationally challenging. Therefore, a Monte Carlo joint-space sampling approach is adopted to obtain an approximate representation of the workspace.

Random joint configurations are generated within the robot's joint limits. For each configuration, forward kinematics is computed using Pinocchio to determine the pose of the end-effector. The translational component of the end-effector pose is then extracted and stored as a point in the workspace.

Result

A total of 50,000 random joint configurations were sampled within the joint limits of the FR3 manipulator. The corresponding end-effector positions were computed using forward kinematics and visualized as a three-dimensional point cloud, providing an approximation of the robot's reachable workspace.

Limitation of Reachable Workspace

Although the reachable workspace identifies all positions that the robot can attain, it does not provide any information about the robot's dexterity or motion capability at those locations.

Two points may both lie within the reachable workspace, yet the robot may exhibit significantly different kinematic behavior at each point. At one location, the robot may move freely in all Cartesian directions, while at another it may operate close to a singular configuration with severely restricted motion.

Therefore, reachable workspace alone is insufficient for evaluating the quality of a robot's workspace. A performance measure is required to quantify how effectively the manipulator can generate motion throughout its workspace.

Manipulability Workspace

Concept

Manipulability workspace extends the reachable workspace by evaluating the robot's kinematic performance throughout the reachable region. Rather than indicating only whether a point can be reached, it quantifies how effectively the manipulator can produce motion in different Cartesian directions at that configuration.

In this work, the Yoshikawa manipulability index is used as a measure of dexterity. Configurations with higher manipulability indicate better motion capability, whereas lower values indicate reduced dexterity and proximity to singular configurations.

Mathematical Definition

The manipulability index is computed using the translational component of the geometric Jacobian, since the objective of this analysis is to evaluate the robot's ability to generate linear motion throughout its workspace.

w = √det(JvJvT)

where,

  • Jv is the translational Jacobian of the end-effector.
  • w is the Yoshikawa manipulability index.

Implementation

The manipulability workspace is generated using the same set of joint configurations employed for the reachable workspace analysis. For each sampled configuration, the translational Jacobian of the end-effector is computed using Pinocchio, and the corresponding Yoshikawa manipulability index is evaluated.

The computed manipulability value is associated with the corresponding end-effector position and visualized using a color map. Regions with higher manipulability indicate better dexterity, whereas lower values highlight configurations with reduced motion capability and proximity to singularities.

Result

The manipulability workspace provides a more informative representation of the robot's operational capability than the reachable workspace alone. While every point in the reachable workspace can be attained, the manipulability distribution reveals regions where the robot can generate motion more effectively and regions where its dexterity is significantly reduced.

Conclusion

This project presented a workspace analysis framework for the Franka Research 3 manipulator using ROS2 and Pinocchio. The reachable workspace identifies all attainable end-effector positions, whereas the manipulability workspace provides additional insight into the robot's kinematic performance throughout the workspace. Together, these analyses offer a comprehensive understanding of the robot's capabilities and provide valuable information for task planning, robot placement, and workspace optimization.
